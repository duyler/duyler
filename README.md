# Duyler
### Event-driven architecture framework

### Get started

#### Install without host php interpreter

```shell
git clone https://github.com/duyler/duyler.git
```
```shell
cd duyler
```
```shell
make build
```

#### Install using local composer

```shell
composer create-project duyler/duyler
```
```shell
cd duyler
```
```shell
make build
```
Open http://localhost/
